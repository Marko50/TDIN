Sync - Normal synchronous call of a time consuming method
       (counting the primes between 2 and a user chosen limit)
	 Choose something about 300000000 (300 million) values.
ASync - The same call performed asynchronously using a delegate.
NewASync - Yet the same call performed asynchronously using the
           new keywords async and await introduced in C# 5.0.
AsyncThreads - Show an async call and the threads where code run.
               Uses a Task as a result and how to get the returned value.
JavaAsync - An equivalent asynchronous call using the features
            of Java 8.
